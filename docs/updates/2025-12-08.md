## How to Use

### Configure OpenAI API Key
Add to your `.env.local`:
```bash
OPENAI_API_KEY=sk-your-api-key-here
```

### Using AI Suggestions
1. Navigate to `/articles/generate/step-1` and add your article samples
2. Continue to step 2 (Topics)
3. Click "Generate Ideas" button
4. AI will analyze your writing style and suggest relevant topics
5. Click any suggestion to add it to your topics list
6. Use the refresh button to get more ideas

### Features
- Personalized suggestions based on your writing samples
- Avoids suggesting topics you've already added
- Refresh functionality for unlimited new ideas
- Error handling with user-friendly messages
